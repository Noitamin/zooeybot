# zooeybot v1.0
Bot for discord shenanigans 

# Requirements
Install the following packages and dependencies BEFORE installing discord.py
- python3.5+
- python3.5-dev
- libffi-dev
- discord.py
```
python3 -m pip install -U discord.py[voice]
```
Put your token in secrets.py
